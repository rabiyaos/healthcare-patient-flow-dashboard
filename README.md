# Hospital Patient Flow Optimization Dashboard

## Problem Statement

Hospitals often face inefficiencies in patient flow, leading to long wait times, overcrowding, and poor resource utilization.

This project analyzes patient data to identify bottlenecks and provide actionable insights to improve hospital operations.

---

## Tools & Technologies

* SQL (Data extraction & analysis)
* Power BI (Dashboard & visualization)
* Python (Data analysis)

---

## Key Analysis Performed

* Patient admission trends by time and department
* Average wait times across departments
* Peak hours causing congestion
* Patient discharge patterns

---

## Key Insights

* Peak admission hours significantly increase wait times
* Certain departments experience consistent bottlenecks
* Delays in discharge contribute to bed unavailability
